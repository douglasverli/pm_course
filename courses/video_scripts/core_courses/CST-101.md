[OPEN WITH GENUINE ENTHUSIASM]

Alright, welcome to Introduction to Programming with Python. I'm Douglas Verli, and I have to tell you – I'm genuinely pumped that you're here.

Now, I get asked all the time: "Do I really need to learn programming? Isn't that just for tech nerds?" And honestly, the answer depends on what you want to do with your life. But even if you think you don't want to be a professional programmer, I'd argue that understanding how code works is becoming as fundamental as understanding how to use email.

Here's why: Everything around you is code. Your phone, your car, your favorite app, the systems that handle your bank account, the algorithms that decide what shows up in your social media feed. Code is everywhere. And more jobs than ever are looking for people who can think like programmers – not necessarily people who code professionally, but people who understand the logic, who can solve problems systematically, who can understand what's possible and impossible.

Plus, programming is just creative problem-solving. It's like solving a puzzle, but the puzzle does something useful when you solve it. It's incredibly satisfying.

So why Python specifically?

Great question. Python is incredible for learning because it reads almost like English. It's not cluttered with weird symbols that make you feel like you're decoding ancient hieroglyphics. You look at Python code and you can kind of understand what it's doing even before you learn the formal rules.

But – and this is a big but – don't let the simplicity fool you. Python is also one of the most powerful programming languages in the world. It's used by Google, Netflix, Instagram, Spotify, NASA, basically every major company. It's the go-to language for data science, artificial intelligence, automation, web development, machine learning – you name it.

So you're learning something that's both beginner-friendly and genuinely powerful. That's rare. Most languages are either easy to learn but limited, or powerful but complicated. Python is both.

Here's what we're going to do in this course:

First, we'll start with the absolute basics. We'll talk about what programming actually is – spoiler alert, it's just giving instructions to a computer, but in a way the computer understands. We'll set up Python on your computer, and you'll write your first program. And yeah, it'll be the classic "Hello, World!" program, but that moment when you run it and see output? That's going to feel really good.

Then we'll learn about variables and data types. Think of variables as labeled containers where you store information. We'll learn about strings (which are words and text), numbers (integers and decimals), and boolean values (true or false). These are the building blocks.

After that comes control flow – and this is where it gets fun. We'll learn about if statements (making decisions in code), loops (doing things repeatedly), and functions (grouping code into reusable chunks). This is where your code starts to do interesting things.

We'll also cover working with data structures – lists, dictionaries, tuples. These are ways of organizing data so you can do powerful things with it.

Then we'll do some real-world stuff. File handling, working with APIs, maybe even a bit of web scraping. And throughout, we'll be building small projects – things that actually do something interesting.

By the end of this course, you won't be a professional programmer. But you'll understand how programming works. You'll be able to write code that solves actual problems. You'll be able to read other people's code and understand it. And you'll have the foundation to learn any other programming language easily, because the concepts transfer.

Now, here's my promise to you: We're going to make this fun. I'm going to break things down. I'm going to explain not just the "how" but the "why." And I'm going to show you real examples of how the stuff we're learning is actually used in the real world.

And here's what I need from you: Don't just watch these videos. Code along. Type out the examples even if you think they're simple. Make mistakes – because honestly, you learn more from debugging your own mistakes than from anything else. Programming is like learning a language – you've got to practice.

So, are you ready to learn how to think like a programmer?

### Video 2: Setting Up Your Environment
**Duration:** 10 minutes
**Learning Objectives:**
- Install Python successfully
- Choose and set up a code editor
- Write and run your first program

---

**SCRIPT:**

[TALK DIRECTLY TO CAMERA]

Okay, before we write any code, we need to set up your workspace. Don't worry – this is actually pretty painless, and I'm going to walk you through it step by step.

First, we need Python installed on your computer. Go to python.org. Download the latest version – as of right now, that's Python 3.something. The "3" is important because Python 2 is old and retired, kind of like a programming language that's gotten its gold watch and gone off to retirement.

Run the installer. During installation, there's usually a checkbox that says "Add Python to PATH." Make sure that's checked. This lets you run Python from anywhere on your computer, which is super convenient. Trust me, you want this.

Once you've installed Python, open your terminal or command prompt and type `python --version`. If you see a version number, you're good. You're installed and ready to go.

Now for a code editor. You could technically write Python code in Notepad, but that would be like building a house with a spoon – technically possible, but why would you do that to yourself?

I recommend Visual Studio Code. It's free, it's lightweight, and it has extensions that make Python development smooth. Download it from code.visualstudio.com, install it, then search in the extensions marketplace for the Python extension by Microsoft. Install that, and you're golden.

Alternatively, if you want something even more beginner-friendly, there's IDLE, which comes with Python, or you could use an online editor like replit.com. No installation needed, you can just start coding in your browser.

Alright, you've got Python installed and you've got an editor. Now let's write our first program.

Open your editor, create a new file called `hello.py`, and type this:

```python
print("Hello, World!")
```

Yes, that's it. That's a complete program.

Now, here's what's happening: `print` is a function – it's a built-in instruction that tells Python to display something on the screen. The text in quotes is what you want to display.

Save the file. In your terminal, navigate to where you saved it, and type `python hello.py`.

You should see `Hello, World!` appear on your screen.

Congratulations. You just wrote code. You're a programmer now. (Okay, you're a programmer who's written one line of code, but still. That's how we all started.)

Now, that was simple, almost too simple. But here's the beautiful part – every program you'll ever write, no matter how complex, is just a bunch of simple steps like that, strung together. It's building blocks. We start with the tiny blocks, then we stack them higher and higher.

In the next video, we're going to talk about variables – how to store information in Python so you can actually do something with it. We'll start doing calculations and manipulations.
